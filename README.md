## 🎮 使用说明

### 基础操作

1. **点击屏幕** - 在点击位置发射烟花
2. **移动鼠标** - 产生彩色粒子轨迹
3. **点击⚙️图标** - 打开/关闭控制面板

### 控制面板

- **烟花大小** - 调节爆炸范围（20-100）
- **粒子数量** - 调节粒子密度（50-200）
- **自动间隔** - 调节自动播放速度（0.5-5秒）
- **祝福文字** - 输入文字并显示
- **自动烟花** - 开关自动播放
- **全屏模式** - 切换全屏显示
- **音效开关** - 开关声音效果

## 🛠️ 技术栈

- **HTML5 Canvas** - 图形渲染
- **JavaScript ES6+** - 核心逻辑
- **Web Audio API** - 音效生成
- **CSS3** - 样式和动画

## 📦 安装使用

### 本地运行

1. 克隆仓库
```bash
git clone https://github.com/yourusername/fireworks-showcase.git
```

2. 进入项目目录
```bash
cd fireworks-showcase
```

3. 用浏览器打开 `index.html`

## 🌐 部署到 GitHub Pages

1. Fork 本仓库
2. 进入仓库 Settings → Pages
3. 选择 Source 为 main 分支
4. 点击 Save
5. 等待 1-2 分钟后访问 `https://你的用户名.github.io/fireworks-showcase/`
